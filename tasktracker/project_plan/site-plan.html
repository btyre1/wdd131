Task Tracker Site Plan

Bryce Tyre

WDD 131-A2

Overview

Purpose

To help users organize their tasks and track daily habits in a simple, interactive way.

Audience

Anyone who wants a simple, web-based tool to manage tasks, assignments, or daily routines.

Dynamic elements

JavaScript will handle adding, completing, and deleting tasks, with arrays and objects storing task data. The page will update dynamically using DOM manipulation, and tasks may be saved in local storage. Users will also be able to filter tasks by all, active, or completed.

Content

Home page

Welcome to Task Tracker! Stay organized and keep your day on track. Task Tracker helps you quickly add tasks, check them off, and stay focused on what matters most. How It Works: Type a task into the box below and click “Add Task.” Your tasks will appear in a list where you can mark them as complete or remove them at any time. Example Tasks: Finish homework, Practice coding, Read for 20 minutes. Button & Label Text: Add Task, Complete, Delete, View All Tasks. Images / Icons Used: White checkmark logo, Check icon for completed tasks, Trash/delete icon.

Tasks

All Tasks: Here you can view every task you've created, including completed and active ones. Use this page to review your progress, organize your list, and remove tasks you no longer need. Task Filters: All Tasks, Active Tasks, Completed Tasks. Section Text: Your Task List, No tasks yet. Add some on the homepage!, Clear Completed Tasks. Images / Icons Used: Check icon for completed tasks, Trash/delete icon, Filter icons.
